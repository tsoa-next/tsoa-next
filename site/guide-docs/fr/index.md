<section class="home-search-section" aria-labelledby="search-summary-title"><h2 id="search-summary-title">Générer OpenAPI de TypeScript sans dérive</h2><p><code>tsoa-next</code> conserve vos contrôleurs, validation d'exécution, itinéraires générés, et OpenAPI sortie alignée de sorte que les documents API restent précis à mesure que votre code évolue.
 </p><p>Les documents ici couvrent la configuration, la génération, les décorateurs, la validation, les téléchargements de fichiers, l'authentification, l'injection de dépendance et la personnalisation des modèles pour Express, Koa, Hapi, et les piles de serveur connexes.
 </p></section>
